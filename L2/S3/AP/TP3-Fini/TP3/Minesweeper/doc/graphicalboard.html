graphicalboard module¶

Ce module fournit essentiellement une fonction de création d’une fenêtre graphique pour jouer au démineur. Il nécessite les modules cell et minesweeper.

  • Création d’une fenêtre de jeu :

    graphicalboard.create(game)[source]¶

    This function creates the graphical board from a game. It also launches the event loop. Thus, this is the only function to run to have a functional graphical board.

    Parameters:game (Minesweeper) – the minesweeper game
    Returns:None
